# The-Isle-JE-Stats

Fetches and overlays dinosaur information over the screen. Made for Jurassic Echos server, information is extracted from their website.

---

### The-Isle-JE-Stats
![TIJEStats Overlay](screenshots/TIJEStats_Overlay.png)

---

## Quick setup

Insert your browser data into the [`fetching`](fetching.json) file. The needed information is:
- **User Agent:** So we don't get flagged for being a bot.
- **Cookie for the website:** So the server knows what information to get.
